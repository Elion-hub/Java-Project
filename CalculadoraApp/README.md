# 🧮 Calculadora en Java (Consola)

Aplicación de consola en **Java** que permite realizar operaciones aritméticas básicas con números enteros y decimales.

## 🚀 Funcionalidades
- ➕ Suma  
- ➖ Resta  
- ✖️ Multiplicación  
- ➗ División  
- 🚪 Salir
